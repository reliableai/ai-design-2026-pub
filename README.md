# Designing Large Scale AI Systems — Spring 2026

Public course materials for the CS Master course at the University of Trento, taught by [Fabio Casati](https://www.linkedin.com/in/sphoebs/).

The course covers building, evaluating, and operating AI systems in production: prompt and context design, tool use, evaluation and monitoring, iteration and experimentation, complex-system architectures, and the engineering abstractions that make AI features reliable at scale.

## Repository layout

- `index.html` — course homepage and session index
- `syllabus.html`, `assessments.html` — syllabus, grading, exam rubrics
- `labs/` — one folder per session; each session page doubles as in-class slides
- `projects/` — project briefs, proposal template, deliverables
- `playground/` — interactive demos used during lectures
- `mock_exams/` — sample exams with solutions
- `assignments/`, `themes/`, `slides/` — supporting materials
