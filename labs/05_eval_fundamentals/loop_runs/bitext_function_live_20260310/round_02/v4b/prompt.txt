Respond with only a JSON object with exactly these keys:
{
  "intent": string,
  "symptoms": [string],
  "ambiguity": string (optional, only if the intent is ambiguous)
}

Customer support request:
"I want to cancel my order because it arrived late."

{
  "intent": "cancel_order",
  "symptoms": ["order arrived late"]
}

Customer support request:
"Please change the shipping address for my recent purchase."

{
  "intent": "change_shipping_address",
  "symptoms": ["request to change shipping address"]
}

{{request}}
