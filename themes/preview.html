Course Description

This course provides a comprehensive introduction to designing, building, and evaluating large-scale AI systems. Students will gain hands-on experience building AI-powered applications and learn rigorous evaluation methodologies essential for deploying AI systems in production.

Learning Objectives

By the end of this course, you will:

  1. Build effective, reliable, maintainable AI systems at scale — solving complex problems by decomposing them in a principled, manageable way.
  2. Use AI to build AI. Leverage AI for development, evaluation, and system management. Understand what this unlocks and where the limits are.
  3. Iterate towards success — rather than optimizing in the dark. Master the science of monitoring, analysis, and estimation.
  4. Master the key conceptual abstractions and middleware we need to brave this new world.
  5. Separate signal from hype. Critically review emerging practices and proposed standards.
  6. Think in systems. Understand which engineering principles carry forward and which new abstractions are essential.

Course Structure

The course spans 12 weeks with two 90-minute lessons per week (24 lessons total), organized into four phases:

Phase 1

Building AI Agents

Lessons 1–3

LLM API calls, context and memory, the agentic loop, tool integration.

Phase 2

Iterating Towards Success

Lessons 4–6

Monitoring, quality estimation, uncertainty quantification, systematic iteration.

Phase 3

Complex Systems at Scale

Lessons 7–11

Councils of agents, system design, middleware, guardrails, frameworks.

Phase 4

Capstone Project

Lessons 12–24

Apply everything: specification, midterm, reviews, working sessions, presentations.
